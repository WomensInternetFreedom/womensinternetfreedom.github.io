About Spinster

Spinster is a woman-centered, federated, social media site in the microblogging format of Twitter.

Join Spinster

Technologies Used

  • React JS

    Spinster's frontend, SoapboxFE, is build in React.

  • Elixir

    The backend Spinster runs, Pleroma, is built in Elixir.

  • ActivityPub

    ActivityPub is a decentralized social networking protocol. It provides a client to server API for creating, updating and deleting content, as well as a federated server to server API for delivering notifications and subscribing to content.
