About FemTube

FemTube is a feminist alternative to Youtube where women and allies can share content without fear of censorship.

Watch FemTube

Technologies Used

  • PeerTube

    PeerTube is a free and decentralized alternative to video platforms.

  • ActivityPub

    ActivityPub is a decentralized social networking protocol. It provides a client to server API for creating, updating and deleting content, as well as a federated server to server API for delivering notifications and subscribing to content.
